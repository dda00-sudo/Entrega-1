# Comandos Principales de Git

| Comando | Descripción | Ejemplo |

| **git init** | Inicializa un nuevo repositorio de Git en el directorio actual. | `git init` |
| **git status** | Muestra el estado de los archivos en el directorio de trabajo y el área de preparación. | `git status` |
| **git add** | Añade archivos del directorio de trabajo al área de preparación (staging area). | `git add archivo.txt` o `git add .` |
| **git commit** | Registra los cambios del área de preparación en el historial del repositorio. | `git commit -m "Descripción del cambio"` |
| **git push** | Sube los commits locales al repositorio remoto (en GitHub). | `git push origin nombre-de-la-rama` |
| **git checkout** | Se usa para cambiar de rama o restaurar archivos del directorio de trabajo. | `git checkout otra-rama` |
